
HOWTO:

1. Export SDK_PATH and BIN_PATH.
Example:
   Your SDK path is ~/Luanode, and want generate bin at ~/ESP32_BIN.
   Do follow steps:
     1>. export SDK_PATH=~/LuaNode_Esp32
     2>. export BIN_PATH=~/LuaNode_Esp32_BIN
   SDK and project are seperate, you can update SDK without change your project.

2. Enter project_template folder, run ./gen_misc.sh.
